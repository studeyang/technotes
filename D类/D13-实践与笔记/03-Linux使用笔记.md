## 一、循环调用接口

```shell
# !/bin/bash

for i in {100..786}
do
  id="sdbc0$i"
  data="{\"ids\":[\"$id\"],\"startTime\":\"2021-07-03 00:00:00\",\"endTime\":\"2021-07-03 23:59:59\"}"
  curl -v -H "accept: */*" -H "Content-Type:application/json" -X POST http://xxx/a -d "$data" >> reconsume.log
done
```

##  三、查看某个进程的线程数

```shell

```
